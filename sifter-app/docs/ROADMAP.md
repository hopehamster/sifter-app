# Roadmap

## Overview

This roadmap outlines the future development plans for Sifter App. It includes upcoming features, improvements, and milestones.

## Current Version

### Version 1.1.0 (2024-04-01)

- Dark mode
- Push notifications
- Location sharing
- Profile customization
- Chat search

- Read receipts
